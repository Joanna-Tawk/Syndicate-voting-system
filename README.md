# Syndicate Online Voting System

This project is an online voting system developed as a university project.

## Technologies
- React.js (Frontend)
- Node.js + Express (Backend)
- MySQL (Database)

## Features
- Role-based access (Monitor, Candidate, Voter)
- Candidate registration
- Secure voting system
- Live monitoring
- Final election results

## How to Run
### Backend
- npm install
- node server.js

### Frontend
- npm install
- npm start
